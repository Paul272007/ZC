# ZC

A utility to compile C easily and faster

## Installation :

- Clone the repository, go into ZC folder and run `./install.sh` with sudo privileges to install ZC

## Usage :

- With no options, the program just compiles the program and executes it without you having to add some options. It also deletes the created executable so your folder isn't filled with random execuatble files everywhere.
  ...

## Uninstalling :

- Simply run `./uninstall.sh` with sudo privileges to uninstall ZC and its configuration
